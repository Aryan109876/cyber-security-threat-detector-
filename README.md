# cyber-security-threat-detector-
This is a Full-Stack AI-powered Cybersecurity Threat Detector designed to identify potential security threats in real-time. Using TensorFlow.js for in-browser AI threat detection, an Express.js backend for log analysis, and a Next.js frontend for visualizing detected threats, this project showcases a powerful combination of AI, security, and full-stack development.
